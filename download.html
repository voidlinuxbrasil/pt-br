Baixe imagens live instaláveis ​​e tarballs rootfs

Todas as imagens live e tarballs rootfs estão disponíveis em:

  • https://repo-default.voidlinux.org/live/current

Esses arquivos também podem ser baixados de outros espelhos, listados na documentação. Basta navegar para live -> current para encontrá-los.

Os requisitos para essas imagens podem ser encontrados na documentação. Uma conexão com a Internet via Ethernet ou WiFi é necessária para a instalação da rede.

Verificando a integridade do arquivo e sua assinatura digital

É altamente recomendável validar a integridade e autenticidade de qualquer imagem baixada ou tarball antes de usá-lo, para garantir que não tenha sido adulterado. Instruções sobre como fazer isso são fornecidas no Manual do Void.

Será necessário baixar o arquivo de checksum e sua assinatura para esta etapa.

x86_64

Além da imagem de linha de comando simples, há um sabor gráfico com o ambiente de desktop XFCE. Outros ambientes gráficos são totalmente suportados pelo Void Linux, mas não são oferecidos como imagens de demonstração/instalação, a fim de diminuir a sobrecarga envolvida nos testes.

Imagens live instaláveis ​​suportam uma instalação local (com os pacotes incluídos) ou uma instalação de rede (os pacotes são baixados do repositório oficial).

Você pode fazer login nessas imagens como anon ou root, e a senha é voidlinux.

Para iniciar o instalador, execute o utilitário void-installer com as permissões apropriadas (ou seja, sudo void-installer).

Para instalar os pacotes para o ambiente desktop, NÃO escolha "instalar da rede" escolha a opção "instalação local".

plataformas arm

As imagens live podem ser gravadas em um cartão SD (ou seja, usando dd) e fornecem um sistema pronto para inicializar. Estas imagens estão preparadas para cartões SD de 2GB. Como alternativa, use os tarballs ROOTFS se desejar personalizar as partições e os sistemas de arquivos.

Conecte-se ao sistema usando um terminal virtual ou SSH e faça login como root com a senha voidlinux.

As instruções específicas da plataforma para essas imagens estão disponíveis na documentação.

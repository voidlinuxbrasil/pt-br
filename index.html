The Void (Linux) / Distribuição

Void é um sistema operacional de propósito geral, baseado no monolítico Kernel Linux. Seu sistema de pacotes permite instalar, atualizar e remover software rapidamente; o software é fornecido em pacotes binários ou pode ser construído diretamente de codigo fonte com a ajuda da coleção de pacotes de fontes XBPS.

Está disponível para uma variedade de plataformas. Os pacotes de software podem ser construídos nativamente ou compilados de forma cruzada através da coleção de pacotes do XBPS.

Siga a gente no Twitter, visite o #voidlinux, nosso canal no IRC da rede libera.chat, e junte-se ao Void Linux subreddit.

Visite o Void build server console para atualizações de status de compilação de pacotes.

Contribua para o projeto Void Linux adicionando e atualizando pacotes e estendendo a documentação. Mais informações podem ser encontradas na documentação.

Não é um FORK!

Void Linux é uma distribuição independente, desenvolvida inteiramente por voluntários.

Ao contrário de trilhões de outras distribuições existentes, Void não é uma modificação de uma distribuição existente. O gerenciador de pacotes e o sistema de compilação do Void foram escritos do zero.

Atualizações Rolling Release (estável)

Void se concentra na estabilidade, em vez de fazer seu usuario de cobaia. Instale uma vez, atualize rotineiramente e com segurança.

Graças ao nosso sistema de compilação contínua, novos softwares são incorporados em pacotes binários assim que as alterações são enviadas para o repositório void-packages.

runit

Usamos runit como o sistema de inicialização e supervisor de serviço.

runit é uma abordagem simples e eficaz para inicializar o sistema com uma supervisão confiável. Consulte o Manual do Void para uma introdução.

Diversidade da biblioteca C

Void Linux suporta as implementações musl e GNU libc, corrigindo software incompatível quando necessário e trabalhando com desenvolvedores upstream para melhorar a exatidão e portabilidade de seus projetos.

XBPS

xbps é o gerenciador de pacotes do sistema nativo, escrito do zero com uma licença BSD de 2 cláusulas.

O XBPS permite instalar/atualizar/remover software rapidamente em seu sistema e apresenta detecção de bibliotecas e dependências compartilhadas incompatíveis ao atualizar ou remover pacotes (entre outros). Consulte o Manual para obter uma visão geral.

xbps-src

xbps-src é o construtor de pacotes xbps, escrito do zero com uma licença BSD de 2 cláusulas.

Isso constrói o software em contêineres através do uso de namespaces do Linux, fornecendo isolamento de processos e montagens de ligação (entre outros). Nenhuma raiz necessária!

Além disso, o xbps-src pode ser compilado nativamente ou com compilação cruzada para a máquina de destino e suporta várias bibliotecas C (glibc e musl atualmente).

Notícias recentes

Novo espelho padrão

Maio 27, 2022

O espelho padrão foi atualizado para https://repo-default.voidlinux.org. Se você deseja especificar um espelho que esteja mais próximo de você geograficamente, você pode encontrar a lista completa de espelhos atualmente ativos aqui.

Aposentadoria do espelho dos EUA

Janeiro 03, 2022

O espelho alpha.us.repo.voidlinux.org foi retirado. Os usuários devem mudar para https://repo-us.voidlinux.org para continuar o serviço fora do centro dos EUA. Como parte do switch, o espelho de nível um dos EUA ganhou TLS e está sendo executado em um host mais confiável.

Todos os colaboradores com PRs em andamento devem fazer o rebase para garantir que a URL mais recente seja refletida na configuração de CI de sua filial.

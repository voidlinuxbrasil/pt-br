Instaladores Live

Void fornece imagens do instalador live contendo um conjunto básico de utilitários, um programa instalador e arquivos de pacote para instalar um novo sistema Void. Essas imagens live também são úteis para reparar um sistema que não consegue inicializar ou funcionar corretamente.

Existem imagens x86_64 para sistemas baseados em glibc e musl. Há também imagens para i686, mas apenas glibc é compatível com esta arquitetura. Os instaladores live não são fornecidos para outras arquiteturas. Os usuários de outras arquiteturas precisarão usar tarballs rootfs ou realizar uma instalação manualmente.

Imagens do instalador

Void libera dois tipos de imagens: imagens base e imagens xfce. Os iniciantes em Linux são encorajados a experimentar uma das imagens xfce mais completas, mas usuários mais avançados podem preferir começar a partir de uma imagem base para instalar apenas os pacotes de que precisam.

Imagens base

As imagens de base fornecem apenas um conjunto mínimo de pacotes para instalar um sistema Void utilizável. Esses pacotes básicos são apenas aqueles necessários para configurar uma nova máquina, atualizar o sistema e instalar pacotes adicionais de repositórios.

Imagem Xfce

A imagem xfce inclui um ambiente de desktop completo, navegador da web e aplicativos básicos configurados para esse ambiente. A única diferença das imagens base são os pacotes e serviços adicionais instalados.

O seguinte software está incluído:

  • Gerenciador de janelas: xfwm4

  • Gerenciador de arquivos: Thunar

  • Navegador: Firefox ESR

  • Terminal: xfce4-terminal

  • Editor de texto simples: Mousepad

  • Visualizador de imagens: Ristretto

  • Outros: Bulk rename, Orage Globaltime, Orage Calendar, Task Manager, Parole Media Player, Audio Mixer, MIME type editor, Application finder

O processo de instalação da imagem xfce é o mesmo das imagens base, exceto que você deve selecionar a fonte local ao instalar. Se você selecionar Rede, o instalador baixará e instalará a versão mais recente do sistema básico, sem nenhum pacote adicional incluído na imagem ao vivo.

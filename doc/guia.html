Guia de instalação

Depois de baixar uma imagem do Void para instalar e preparar sua mídia de instalação, você está pronto para instalar o Void Linux.

Antes de iniciar a instalação, você deve determinar se sua máquina inicializa usando BIOS ou UEFI. Isso afetará como você planeja partições. Consulte Notas de particionamento para obter mais detalhes.

Os seguintes recursos não são suportados pelo script do instalador:

  • LVM
  • LUKS
  • ZFS

Inicializando

Inicialize sua máquina a partir da mídia de instalação que você criou. Se você tiver RAM suficiente, há uma opção na tela de inicialização para carregar a imagem inteira na RAM, o que levará algum tempo, mas acelerará o restante do processo de instalação.

Depois que a imagem live for inicializada, faça login como root com a senha voidlinux e execute:

# void-installer

As seções a seguir detalharão cada tela do instalador.

Teclado (Keyboard)

Selecione o mapa de teclas para o seu teclado; teclados "qwerty" padrão geralmente usam o mapa de teclas "us". Mas nós brasileiros usamos o abnt2.

Rede (Network)

Selecione sua interface de rede principal. Se você optar por não usar DHCP, será solicitado a fornecer um endereço IP, gateway e servidores DNS.

Se você pretende usar uma conexão sem fio durante a instalação, pode ser necessário configurá-la manualmente usando wpa_supplicant e dhcpcd manualmente antes de executar o void-installer.

Fonte (Source)

Para instalar os pacotes fornecidos na imagem de instalação, selecione Local. Caso contrário, você pode selecionar Network para baixar os pacotes mais recentes do repositório Void.

Aviso: Se você estiver instalando o ambiente de desktop a partir da imagem do xfce, você DEVE escolher Local para a Source!

Nome da Máquina (Hostname)

Selecione um nome de host para o seu computador (todo em letras minúsculas, sem espaços).

Localidade (Locale)

Selecione suas configurações de localidade padrão. Esta opção é apenas para glibc, já que musl não suporta locales no momento.

Fuso horário (Timezone)

Selecione seu fuso horário com base nas opções de fuso horário padrão.

Senha do Root (Root password)

Digite e confirme sua senha de root para a nova instalação. A senha não será mostrada na tela.

Conta de usuário (User account)

Escolha um login (padrão void) e um nome descritivo para esse login. Em seguida, digite e confirme a senha para o novo usuário. Você será solicitado a verificar os grupos para esse novo usuário. Eles são adicionados ao grupo wheel por padrão e terão acesso sudo.

Os nomes de login têm algumas restrições, conforme descrito em useradd(8).

Bootloader

Selecione o disco para instalar um bootloader quando o Void for instalado. Você pode selecionar nenhum para pular esta etapa e instalar um carregador de inicialização manualmente após concluir o processo de instalação. Se estiver instalando um bootloader, você também será perguntado se deseja ou não um terminal gráfico para o menu GRUB.

Partição (Partition)

Em seguida, você precisará particionar seus discos. O Void não fornece um esquema de partição predefinido, portanto, você precisará criar suas partições manualmente com o cfdisk(8). Você será solicitado com uma lista de discos. Selecione o disco que você deseja particionar e o instalador iniciará o cfdisk para esse disco. Lembre-se de que você deve gravar a tabela de partições na unidade antes de sair do editor de partições.

Se estiver usando UEFI, é recomendável selecionar GPT para a tabela de partições e criar uma partição (geralmente entre 200 MB-1 GB) do tipo Sistema EFI, que será montada em /boot/efi.

Se estiver usando o BIOS, é recomendável selecionar MBR para a tabela de partição. Usuários avançados podem usar o GPT, mas precisarão criar uma partição BIOS especial para o GRUB inicializar.

Consulte as Notas de Particionamento para obter mais detalhes sobre como particionar seu disco.

Sistemas de arquivos (Filesystem)

Crie os sistemas de arquivos para cada partição que você criou. Para cada partição, você será solicitado a escolher um tipo de sistema de arquivos, se deseja criar um novo sistema de arquivos na partição e um ponto de montagem, se aplicável. Quando terminar, selecione Concluído para retornar ao menu principal.

Se estiver usando UEFI, crie um sistema de arquivos vfat e monte-o em /boot/efi.

Revisar configurações

É uma boa ideia revisar suas configurações antes de continuar. Use a tecla de seta para a direita para selecionar o botão de configurações e pressione enter*. Todas as suas seleções serão mostradas para revisão.

Instalar (Install)

Selecionar Instalar no menu iniciará o instalador. O instalador criará todos os sistemas de arquivos selecionados e instalará os pacotes básicos do sistema. Em seguida, ele gerará um initramfs e instalará um carregador de inicialização GRUB2 na partição inicializável.

Todas essas etapas serão executadas automaticamente e, após a instalação ser concluída com êxito, você poderá reinicializar em sua nova instalação do Void Linux!

Pós-instalação

Após inicializar em sua instalação Void pela primeira vez, execute uma atualização do sistema.

$ sudo xbps-install -Suv
